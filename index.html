Welcome to Memphis

Memphis is a C# operating system built by the creator of ShiftOS C#, Michael VanOverbeek. It's purpose is not to make money, or to get famous. It's simply to gain more knowledge of the .NET framework and to increase development skills.

Features

  • One of the fastest booting OSes I've seen - thanks to the C# Open Source Managed Operating System (Cosmos)
  • Working file browser!
  • Easy as heck to install. Seriously. Grandma could do it.

Future features

These are some plans for the future:

  • ShiftOS compatibility
  • Roslyn C# compiler built directly into the kernel.
  • Multiple languages (Polish is planned.)
  • Appscape Package Manager
  • Full Windows Forms port
